About ZWidgets
            ×
        

        

            
ZWidgets is UI component library based on HTML5 and MVVM for quickly building powerful and high-performance web applications.

            
    
                
  • Fast & Light
                    
    Quickly build lightning-fast applications with the simple codes, no dependence on other frameworks.
    
                
    • 
                
  • Easy to use
                    
    Based on the MVVM and OOP, if you know basic JavaScript and HTML, you can use it.
    
                
    • 
                
  • Extensible
                    
    Rich and complete components with powerful extensions, make it easy to customize.
